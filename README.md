 *AirBnB Listing Analysis: Paris-----------------------------------------------------------------------------------------------------

A data analysis project to identify trends in pricing and host growth in Paris, specifically looking at the impact of the 2015 regulations.

 Project Concept----------------------------------------------------------------------------------------------------------------------
The goal is to analyze how the AirBnB market in Paris responded to legal changes. The project includes data cleaning, visualization, and neighborhood-based pricing analysis.

 Key Insights------------------------------------------------------------------------------------------------------------------------------------------
* **Regulation Impact:** Clear shift in new host registrations after 2015.
* **Neighborhood Analysis:** Identified **Elysée** as the most expensive area.
* **Price Trends:** Average prices remained stable despite new rules.

 Tools Used------------------------------------------------------------------------------------------------------------------------------------------
* Python (Pandas, Matplotlib, Seaborn).
* Jupyter Notebook for interactive analysis.
